Blog_1 - Hello World

11/04/2019
Picking the subject matter for what I wanted to discuss on the first blog was harder than I thought. At the end, I wanted to pay homage to the mtcars dataset, from 1974 Motor Trend US magazine. A dataset that is frequently used in the data science community to illustrate data analysis concepts. I have used mtcars dataset quite a bit to test tidying, transforming, visualizing, modeling and even machine learning over the years.

Depending on the the question you want answered, and the data type you have on hand, there are many workflow models to choose from when analyzing a data. A typical data analysis model follow a frame work similar to what you see on Fig1.

We are going to use the mtcars dataset and go thru the first three stages of analysis - Asking the question, getting and preparing the data and explorring and visualizing the data.

Formualting the Question

The first order of business before starting any analysis is to prepare a set of questions we want to ask our data. The questions should be as detailed as possible, because generic questions tend to not yield helpful answers from data. In practice, more questions arise once we start digging into the data and know more about what other hidden answers can be extracted from data. For this blog analysis of mtcars dataset, we will ask three questions and limit the questions to those for brevity.

Let us set a scenario to help frame the questions: Suppose you want to puchase a vehicle and walk onto a used car sales lot that has 32 used cars. Each car has its own features such as miles per galon, number of cylinders, trasmission type etc. In the mtcars dataset there are 11 features listed for each car. To shorten the list to few cars you want to take a look at, you have the following three questions in mind:

Your objective is to buy a car that consumes the least amount of gas per mile. So, the “mpg” is the feature we will focus on to formulate our the questoins.
1/ You don’t want to spend too much money on gas, so you want to know “which car has the maximum and which car has the minimum mpg for all the cars in the lot”.
2/ The cars on the lot have either 4, 6 or 8 cylinders. You want to know “what the average mpg for each cylinder type?”
3/ Lastly, you want to know for each cylinder group “which cars are the most fuel effcient - the top 5%”

Hopefully, the three questions will narrow down the list for your choice!

Although the matcars dataset is in tidy format and has no missing values to be concerned about, it is still row and need further transformation to help us answer the questions raised. I use tidyverse and base R functions to get the data ready for analysis. The dataset is relatively small, it contains 32 observations and 11 features. But the same process would have been sued if the data contained hundreds of thousands of observations and features.

Transofrm and Explore the data

Next we will transform the original mtcars dataset that will answer questions raised. We will generate 4 datasets targeting the questions raised. The first one help us understand the impact of a car weight on mpg, and the other three target each question. I will cover feature relationships and feature selection that helps us understand the data on another blog

The first data set (A - df) simply adds the car model name as a feature. This is important because we want to use the car name to label to data points. The second dataset (B - mx_mpg) extracts the which cars have maximum and which cars have minimum mpg from all the cars. The third dataset (C - avt_mpg) calculates the average mpg for each cylinder group, the cars in the lot have either 4, 6 or 8 cylinders. And the last datset (D - top_5) extracts the top 5%, most efficient, cars in each cylinder group.

library(dplyr)  #load the library for transofrorming data

# A. add the model name as a variaiable
df <- mtcars %>% mutate(car_model = row.names(.))

# B. get the maximum and minimum mpg for all cars
mx_mpg <- df %>% slice(which.max(mpg), which.min(mpg)) %>% mutate_each(list(as.factor), 
    cyl)

# C. Get the avg mpg for each cylinder type
avg_mpg <- df %>% group_by(cyl) %>% summarize(avg_wt = mean(wt), avg_mpg = mean(mpg)) %>% 
    mutate_each(list(as.factor), cyl) %>% mutate(note = paste0(round(avg_mpg), 
    " mpg"))

# D. In each cyl group, cars whose mpg is in the top 5%
top_5 <- df %>% group_by(cyl) %>% filter(mpg > quantile(mpg, probs = 0.95)) %>% 
    ungroup() %>% mutate_each(list(as.factor), cyl)

As you can see in the code, I used dplyr verbs such as (gropu_by, summarize, mutate, filter) and statical functions such (max, min, mean, quantile) and weaved them together to generate additional the four new dataset that help answer the question we raised. This part of the analysis can be done many different ways and it is both an art and a science.

Lets us visualize the answers to the question

After importing, transforming and exploring the data, we want to visualize and communicate the result. We can generate documentation to report the findings, but for now let us use graphic plots to keep it brief. Visual communication is very powerful because it will not only show the point of interest, but gives reference that is helpful when you compare multiple data points.

We are going to use the four dataset created and grammar of graphic plot v2 [ggplot2]https://ggplot2.tidyverse.org/ to generate the visuals that answer the questions.

A. Base plot - shows the impact of weight on mpg

library(ggplot2)

gg <- ggplot(df, aes(x = wt, y = mpg)) + geom_point() + theme(legend.position = c(0.85, 
    0.85))

gg + geom_point(data = top_5, aes(x = wt, y = mpg, fill = cyl), size = 4.5, 
    shape = 21, color = "black", size = 5, stroke = 1) + geom_text(data = top_5,

Summarizing the result of the analysis

It is clear from the graphic visualization that Toytoa Corola is the most fuel efficient car with 33.9 mpg, and Cadillac Fleet wood is the least efficient with at 10.4 mpg. It is also clear that the heavier the car is the higher the gas consumption. We can also see that the average the fewer the number of cylinders in a car, the higher average mpg. Totyta Corolla, Hornet 4 Drive and Pontiac Firebird are in the top 5 percentile of each cylinder group. With this information on hand, you can make data driven decision and pick a car to take for test drive.

Take away

On this blog, we took a row data, transform and applied statistics to summarize and extract the answers to our questions. We generated plots to visualize the result. We focused on single feature (mpg) out of eleven, and small dataset to illustrate the data science process, and how to extract value we are seeking. The same process can be followed to a large dataset with thousands of features and observations.
